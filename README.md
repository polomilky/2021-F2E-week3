## 系統說明
- 專案運行方式是使用: 
```sh
yarn serve
```
## 資料夾說明
- assets: 圖片
- components: 元件
- plugins: 裡面有 vuetify.js
- router: 配置路由
- store: vuex
- utils: leaflet 產生地圖 / marker function
- views: 頁面

## 使用技術
- Vue Cli
- vuetify
- Axios
